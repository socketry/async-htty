We already have a great UI engine.

Let's just use the browser.

# Translation

優れたUIエンジンはすでに手元にある。
ブラウザを使えばいい。

---

Decades of work have gone into making browsers great at rich UI. Layout engines, font rendering, image decoding, animations, accessibility, internationalisation — all of it, already solved, and standardised across vendors.

The pragmatic move is to *use* that. Don't reinvent layout in the terminal. Put a browser on the client side, and let it do what browsers are already good at.

Every compromise from a minute ago — character cells, escape sequences, terminal-specific quirks — becomes someone else's problem. Specifically, the browser's problem.
