TUIs force compromises.

# Translation

TUIにはどうしても妥協がつきまとう。

---

Anyone who has built a terminal user interface knows this feeling. The terminal is a fantastic tool — a direct interface to the system, composable with pipes, scriptable. But the moment you want to show the user anything richer than text, you hit walls.

Three big ones.
